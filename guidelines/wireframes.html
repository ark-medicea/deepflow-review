Entry: Double-click a leaf node in Graph View, or click "Open" in the detail panel. The graph zooms into the node and transitions to Workspace.

Key interaction: Chat and canvas are linked. "Insert into canvas" in chat → inserts AI-generated content at cursor position. "Replace selection" overwrites highlighted text. All AI edits tracked in History tab.
Agent tasks: If assigned to an AI agent, the canvas is read-only while the agent is running (showing streaming output). Once complete, user can edit the output and change status to "Complete" or send back to agent with feedback via chat.

Layout Summary — Option B

Fixed panels (always present):
— Sidebar: 56px (icon-only, always collapsed in Option B)
— Chat Panel: 320px (always visible, contextual to current view/task)

Variable panels:
— Top Bar: 44px height, spans remaining width
— Hub Bar / Status Bar: 36px, present in Graph + Workspace views
— Detail Panel: 360px, present in Graph + Workspace views. Overlay in List view. Absent in Dashboard/Gallery/Settings

Content area widths (of 1200px total):
— With detail panel: 1200 − 56 − 320 − 360 = 464px canvas
— Without detail panel: 1200 − 56 − 320 = 824px content
— All panels resizable via drag handles (chat: 280–400px, detail: 320–480px)

Chat context per view:
— Dashboard: general ("What needs attention?")
— Gallery: project-level ("Create from template")
— Graph: workflow-level ("Show blocked tasks")
— Workspace: task-level ("Summarise stress test results", "Insert into canvas")
— Settings: config-level ("Change notification preferences")

Graph → Workspace transition:
— Double-click leaf node in Graph → enters Workspace for that task
— Middle pane switches from React Flow canvas to content editor (markdown / embed / agent log)
— Same shell, same panels, different middle content

🔗 Share points (growth):
— Top Bar (Graph/Workspace): "Share" button → copy link, invite collaborator, export as image
— Hub Bar (expanded): "Share progress" → shareable snapshot URL with stats + graph thumbnail
— Gallery project cards: ⋮ menu → Share project link, Publish as template
— Workspace canvas: Share output as read-only link (task deliverable sharing)
— Chat action chips: AI suggests "Share with stakeholders" when outputs are ready
— Shared link experience: Non-users see a read-only branded view (graph, progress, output) with "Sign up to collaborate" CTA
